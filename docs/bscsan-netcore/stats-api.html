🎯Stats API Endpoints

Notes

Endpoints with 🅿🆁🅾 are under the API Pro subscription. To upgrade your API plan, browse through the BscScan APIs page.

//imports
using BscScan.NetCore.Contracts;

//inject
private readonly IBscScanStatsService _bscScanStatsService;

🏷Get Total Supply of BNB on the BNB Smart Chain​

Returns the current amount of BNB in circulation.​

  • Request
  • Response
  • ...
//Sample Request 🍎
 var totalSupplyOfBnb = await _bscScanStatsService.GetTotalSupplyAsync();
//will return a new object TotalSupply with values: 🍊
var totalSupplyOfBnb = new TotalSupply
            {
                Status = "1",
                Message = "OK",
                Result ="22573626345184400000000000"
                }
            };
