🎯Blockchain API Endpoints

Notes

Endpoints with 🅿🆁🅾 are under the API Pro subscription. To upgrade your API plan, browse through the BscScan APIs page.

//imports
using Cash.NetCore.Contracts;
using Cash.NetCore.Models.Request.BlockChain;

//inject
private readonly IBlockChainService _blockChainService;

💰Get Block Count​

Returns the number of blocks in the longest blockchain.​

  • Request
  • Response
  • ...
//Sample Request 🍎
var blockCount = await _blockChainService!.GetBlockCountAsc();
//will return a current blockCount: 🍊
var blockCount = 733366;
🍌 For more information Check out Blockchain - Get Block Count

  • Response
  • ...
