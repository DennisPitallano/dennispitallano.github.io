### üí∞Get balance for a single address.
#### Returns an object with confirmed and unconfirmed balance associated with an address.

<Tabs>
  <TabItem value="request" label="Request" default>

```jsx
//Sample Request üçé
var electrumXBalance = await _electrumXService!.GetElectrumXBalanceAsync(
            "bitcoincash:qr69kyzha07dcecrsvjwsj4s6slnlq4r8c30lxnur3");
```

  </TabItem>
  <TabItem value="response" label="Response">    

```jsx
//will return a current BalanceInfo: üçä
var electrumXBalance = new BalanceInfo
        {
            Success = true,
            Balance = new Balance{
            Confirmed = 1600,
            Unconfirmed = 0
            }
        };
```
  </TabItem>

### üí∞Get balances for an array of addresses.
#### Returns an array of balanes associated with an array of address. Limited to 20 items per request.

  <TabItem value="request" label="Request" default>

```jsx

                 "bitcoincash:qzs02v05l7qs5s24srqju498qu55dwuj0cx5ehjm2c"
```
